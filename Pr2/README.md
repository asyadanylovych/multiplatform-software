# Практична робота №2: Кросплатформний калькулятор викидів

Цей проєкт присвячений розробці мультиплатформного застосунку для розрахунку валових викидів шкідливих речовин (суспендованих твердих частинок) при спалюванні палива.

## Технології
* **Мова:** Kotlin
* **Фреймворк:** Compose Multiplatform (Desktop + Android)
* **Середовище:** IntelliJ IDEA (для Windows), Android Studio (для мобільної версії)

## Функціонал
Застосунок дозволяє розраховувати:
1. **Показник емісії (k)** за формулою (2.2) з урахуванням технологічних параметрів котла та ефективності фільтрів.
2. **Валовий викид (E)** за формулою (2.1) для вугілля та мазуту.
